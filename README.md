Trading Behavior & Sentiment Analysis

Project Overview :

This project analyzes trader behavior and performance using historical trade data combined with market sentiment (Fear & Greed Index).

The objective is to:

Analyze trading activity and performance

Compare behavior across sentiment regimes (Fear vs Greed)

Segment traders based on activity and risk exposure

Propose actionable strategy insights

All outputs, visualizations, and results are already included in the Jupyter Notebook.

├── historical_data.csv
├── fear_greed_index.csv
├── analysis_notebook.ipynb
└── README.md

Clone the Repository (or download files):

git clone <your-repo-link>
cd <project-folder>

Install These libraries:

pip install pandas matplotlib numpy scipy jupyter

Step 3: Run All Cells
